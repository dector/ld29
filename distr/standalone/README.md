Run in Windows
==============

`BlueWaves.bat`

Run in Linux/MacOS
==================

`./BlueWaves.sh`

Run everywhere
==============

`java -jar blue_waves.jar`

Controls
========

  - `Mouse` - aim, make photo;
  - `+`/`-` music volume up/down;
  - `M` - mute music;
  - `F` - toggle fullscreen;
  - `R` - (on menu) enter in relax mode.
